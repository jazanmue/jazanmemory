ذاكرة جازان - Jazan Memory

طريقة التشغيل:
1) افتحي ملف index.html لعرض الموقع.
2) افتحي admin.html للدخول إلى لوحة الإدارة.
3) كلمة المرور الافتراضية: 1351
4) من لوحة الإدارة يمكن إضافة وثيقة، صورة، أو فيديو، وتعديل العنوان والتاريخ والموقع والوصف والتصنيف.
5) الإضافات تحفظ داخل متصفحك عبر LocalStorage. للاستضافة على GitHub Pages استخدمي زر "تصدير البيانات JSON" للاحتفاظ بنسخة، ثم استيرادها لاحقًا.

ملاحظة مهمة:
هذه نسخة ثابتة بدون قاعدة بيانات. لو أردتِ أن تكون الإضافات محفوظة على الإنترنت لكل الأجهزة، يلزم ربط الموقع لاحقًا بـ Firebase أو Supabase أو لوحة إدارة خلفية.
